# AGCN
Pytorch implementation of paper 'Adaptive Graph Convolutional Networks for Medical Image Segmentation'.
Our work has been accepted by EMBC 2023.



![overview](https://github.com/11yxk/AGCN/blob/main/AGCN.png)
